开发工具

Jdk

红帽编译的windows版本jdk ：java-1.8.0-openjdk-1.8.0.212-3.b04.redhat.windows.x86_64

不需要手动配置环境变量，因为已经自动配置了环境变量

Maven

版本：apache-maven-3.6.2

自建仓库目录：maven-repository

MAVEN_HOME E:\Java\apache-maven-3.6.2

path %MAVEN_HOME%\bin

setting 配置本地仓库和远程仓库

Gradle

版本：gradle-6.7.1-bin

自建仓库目录：maven-repository

下载地址： https://gradle.org/releases/

环境变量： GRADLE_HOME E:\Java\gradle-6.7.1

GRADLE_USER_HOME E:\Java\maven-repository

path %GRADLE_HOME%\bin

在Gradle安装目录下的 init.d 文件夹下，新建一个 init.gradle 文件，里面填写以下配置。

Idea

历史版本下载地址：https://www.jetbrains.com/idea/download/other.html

2020.1.3或之前版本，后面的版本应用此方式无法破解

破解： https://www.cnblogs.com/yinyueyu/p/11451393.html

  • Alibaba Java Coding Guidelines 阿里代码规范
  • BugCleaner 公司测试代码校验
  • Free Mybatis plugin Mybatis 跳转插件
  • Key Promoter X 快捷键指引
  • Lombok 注解编程
  • Maven Helper 依赖关系管理
  • Rainbow Brackets 括号增强
  • Translation 翻译工具

安装: WebStorm-2018.2.1.exe

Mysql

版本：Mysql5.7.23

安装参考：https://www.cnblogs.com/kunjian/p/11202636.html

1.检查及卸载原版本

  1. 右击我的电脑/计算机，点击管理，打开计算机管理，依次选择服务和应用程序——服务。
  2. 使用快捷命令：同时按下win+r键，在运行窗口中输入services.msc，即可打开服务。

打开系统服务后页面如下，可以看到我的电脑中已经安装有MySQL，下面我将其卸载，若你的电脑没有MySQL服务则不需要操作此步。
